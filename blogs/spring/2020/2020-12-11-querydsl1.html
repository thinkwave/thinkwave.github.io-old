# Number 비교 연산자

  1. age <= 30
    user.age.loe(30);
    1
  2. age < 30
     user.age.lt(30);
    1
  3. age >= 30
    user.age.got(30);
    1
  4. age > 30
    user.age.gt(30);
    1

# Equal 비교 연산자

  1. username == 'user1'
    user.username.eq("user1")
    1
  2. username != 'user1'
    user.username.eq("user1").not()
    1
  3. username != 'user1'
    user.username.ne("user1")
    1
  4. username != null
    user.username.isNotNull()
    1

# 범위 비교 연산자

  1. age between 30 and 35
    user.age.between(30, 35)

  2. age in(30, 31, 32, 33, 34, 35)
    user.age.in(30, 31, 32, 33, 34, 35)

  3. age not in( 30, 31, 32, 33, 34, 35)
    user.age.notin(30, 31, 32, 33, 34, 35 )
    1

# 기타 비교 연산자

  1. like 'user%' user로 시작하는 username 모두 검색

# 페이징

offset과 limit로 페이징을 할 수 있다. offset은 0 부터 시작하는 페이지 이고 limit는 한 페이지에서 몇 건을 읽어 올지를 지정한다.

List<User> users = queryFactory
    .selectFrom(user)
    .orderBy(user.age.asc())
    .offset(1)  // 2 페이지
    .limit(20)  // 20건
    .fetch();

# Case

select 절에서 case 문을 사용하는 방법을 알아 보자. when, then, otherwise 메소드를 체이닝해수 사용한다.

List<String> ageStr = queryFactory
    .select(user.age
        .when(31).then("서른하나")
        .when(32).then("서른둘")
        .otherwise("비밀")
    )
    .from(user)
    .fetch();
